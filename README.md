# podushka-tools
Client-side fabric kotlin mod that adds some tools for personal use.
## Features:
- Command menu
- Soon there will be more features :D

## Requirements

- Minecraft 1.21.11
- Fabric Loader 0.18.1
- Fabric API
- Fabric Language Kotlin

The mod stores its config in:
